<static> parse(date) → {date}

Parses the input into a date. The date will always be set to the time 00:00:00 GMT+0. This function parses strings of the format YYYY-MM-dd where the first M or d is optional if it is 0.

Parameters:
Name Type Description
date date | string

The date to parse to a timeless date object. If date, the time part will simply be set to the specification. If string, it will be parsed to the given date.
